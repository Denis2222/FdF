# FdF

**Ce projet consiste à créer graphiquement la représentation schématique d’un
terrain en relief.**

Compile
```
make
```

Launch
```
./fdf ./map/NOAA_ETOPO_HIMALAYA.OCEAN1.XL.fdf
```
